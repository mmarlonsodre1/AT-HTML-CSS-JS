História

O jogo tem uma história linda, tudo começou entrando no Instituto Infnet.

Apareceu uma coisa chamada AT, e meio que me obrigou a fazer esse jogo.

Estrutura

Utilizei algumas tecnologias que não tenho dominio algum.

O HTML para criar os elementos das páginas, o CSS para mudar o estilo das páginas e o JavaScript para fazer determinada interação.

Desenvolvimento

O projeto foi bem legal, desenvolver uma área que não tinha interesse e ver que é fundamental.

O desenvolvimento foi produtivo, pois consegui montar algumas páginas sem ter nem algo como exemplo, mas fazer do zero.
